ヘルプ@@Window@@を使う

このセクションでは:
  • トピックを見つける
  • 検索のヒント

トピックを見つける

この@@Window@@にヘルプ情報を表示するには、目次 サイドバーの トピックをクリックしてください。次に、+ アイコン△マークをクリックしてそのトピックを展開表示してください。

ヘルプトピックを検索するには、検索 バーに関連するトピックの単語やフレーズを入力します。 検索 サイドバーに検索結果が表示されるので、それをクリックしてください。 検索結果が何も表示されないときは、入力するフレーズを少なくするか、異なる単語の組み合わせを試してみてください。 目次を再びサイドバーに表示するには、検索 サイドバーの閉じるボタンをクリック、 検索 バーに入力したものを消去、または Esc を押下してください。

あなたの読んでいるヘルプトピックに関連した情報を表示するには、いつも Web ページを見ている時と同じように、ヘルプ@@Window@@内のリンクをクリックしてください。 また 戻る ボタンと 進む ボタンを使用して、 すでに訪れたページ間を前へ戻ったり先へ進んだりすることもできます。

検索のヒント

探しているものが見つからないときは、以下のヒントが助けになります:

  • これは一般的なインターネット検索ではありません。 &brandShortName; に関係のあるものの検索かどうか確認してください。
  • 特定しすぎないように、検索範囲を広げてみてください。 検索しようとしている単語は違う表現ができるかもしれません。
  • 検索が広範囲に及ぶものは避けてください。例えば、"ページ" のような共通の単語で検索すると、必要以上に多くのトピックを返します。
